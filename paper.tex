\begin{abstract}
The window between zero-day vulnerability disclosure and weaponized exploitation has collapsed from 745 days down to 44 days, rendering signature-based intrusion detection systems (IDS) obsolete against novel attack vectors. Existing machine learning approaches either strip payload contents into coarse flow statistics or suffer from the quadratic $O(T^2)$ computational complexity of Transformer architectures on raw byte streams. In this paper, we present the \textbf{Sovereign Byte Firewall}, an autonomous, tokenization-free network anomaly detection framework powered by Mamba-2 Selective State Space Models (SSM) and Extreme Value Theory (EVT) Peak-Over-Threshold statistical modeling. We trace the complete evolutionary development of the system from initial Transformer baselines through negative typicality experiments, to native C++/CUDA-accelerated Mamba-2 state space architectures with v2 stateful cross-packet masking. Evaluated on over 1.64 Gigabytes of enterprise network traffic across 24 adversarial attack families, our system achieves a \textbf{0.9788 Calibration ROC-AUC}, a \textbf{94.6\% known attack detection rate}, and a \textbf{0.000\% False Positive Rate (0 false alarms out of 34,043 held-out benign windows)} under EVT tail bounds, while detecting \textbf{28.4\% of novel zero-day attack windows}. The model operates with $<2\,\text{ms}$ per-window inference latency, providing an air-gapped, sovereign defense mechanism against modern zero-day threats.
\end{abstract}

\section{Introduction}
\IEEEPARstart{N}{etwork} perimeter defense is undergoing a fundamental paradigm shift. Recent cybersecurity data indicates that adversarial AI tools have accelerated the discovery and weaponization of zero-day exploits, reducing the average exploitation window from 745 days in FY2022 to 44 days in FY2026. Traditional security infrastructure fails against this accelerated threat landscape:

\begin{enumerate}
    \item \textbf{Signature-Based IDSs} (e.g., Snort, Suricata) rely on explicit CVE rule databases. They exhibit $0\%$ detection capability against novel or obfuscated zero-day payloads.
    \item \textbf{Flow-Based Machine Learning} models (e.g., XGBoost on NetFlow) aggregate traffic into tabular summaries, discarding packet payload contents where zero-day exploit strings (such as shellcode gadgets or deserialization payloads) reside.
    \item \textbf{Transformer-Based Sequence Models} capture byte-level protocol syntax but suffer from $O(T^2)$ quadratic time and memory complexity, creating severe latency bottlenecks on high-throughput perimeter links.
\end{enumerate}

To bridge this gap, we introduce the \textbf{Sovereign Byte Firewall}, a tokenization-free, unsupervised byte-level anomaly detection system. Our framework leverages the linear-time $O(T)$ recurrence of Mamba-2 Selective State Space Models (SSMs) to learn the underlying information-theoretic grammar of normal network traffic. Combined with Extreme Value Theory (EVT) statistical modeling, our system establishes provable upper bounds on false positives, enabling autonomous zero-day protection on air-gapped perimeter infrastructure.

\section{Related Work \& Scientific Foundation}

\subsection{Tokenization-Free Byte Sequence Modeling}
Recent research in network anomaly detection demonstrates that operating directly on raw byte streams ($0 \dots 255$) avoids the fragility and latency of protocol-specific parsers. By modeling raw bytes, neural networks implicitly learn protocol framing, header boundaries, and state transitions without human feature engineering.

\subsection{Selective State Space Models (Mamba-2)}
Mamba-2 replaces traditional Multi-Head Self-Attention with a hardware-aware selective state space layer. Given an input sequence $x$, Mamba-2 maintains a hidden state $h(t)$ governed by continuous-time state parameters $(\mathbf{A}, \mathbf{B}, \mathbf{C}, \Delta)$:
\begin{equation}
    h'(t) = \mathbf{A} h(t) + \mathbf{B} x(t)
\end{equation}
\begin{equation}
    y(t) = \mathbf{C} h(t) + \mathbf{D} x(t)
\end{equation}
Discretized via matrix exponential zero-order hold (ZOH) with step size $\Delta$, Mamba-2 achieves $O(T)$ linear-time evaluation during sequence processing and $O(1)$ constant-time recurrent updates during streaming packet scoring.

\subsection{Architectural Trade-Offs: Transformer vs. Mamba-2}
Initial iterations of our system utilized a Transformer self-attention backbone (\texttt{NetworkBytePatcher}, $d_{\text{model}}=128$). However, Transformer attention maps incurred a quadratic $O(T^2)$ time and VRAM bottleneck as window length $T$ scaled beyond 256 bytes, requiring full $O(T)$ KV-cache expansion during inference. Transitioning to Mamba-2 compressed historical context into a compact state vector $h(t) \in \mathbb{R}^{d_{\text{model}} \times d_{\text{state}}}$, yielding linear $O(T)$ sequence training and constant $O(1)$ streaming inference latency under sub-2ms per-window execution budgets.

\subsection{Information-Theoretic Surprise vs. Typicality}
Anomaly scoring relies on negative log-likelihood surprise $S(x)$ over window $x = (x_1, x_2, \dots, x_T)$:
\begin{equation}
    S(x) = -\frac{1}{T} \sum_{i=1}^{T} \log_2 P(x_i \mid x_{<i}) \quad [\text{bits/byte}]
\end{equation}

During system development, we evaluated two-sided typicality scoring $|S(x) - \mu_{\text{benign}}|$ (based on information-theoretic typical set hypothesis). Empirical evaluation revealed that typicality degraded Calibration ROC-AUC from $0.8038$ down to $0.6232$ due to multi-modal enterprise traffic variance, proving that one-sided upper surprise $S(x)$ is mathematically superior for network payload anomaly detection.

\subsection{EVT Tail Fitting \& Peak-Over-Threshold}
To convert raw surprise scores into reliable operational alerts without manual tuning, we apply Extreme Value Theory (EVT) via the Peak-Over-Threshold (POT) method. According to the Pickands-Balkema-de Haan theorem, excesses above a high quantile threshold $u$ follow a Generalized Pareto Distribution (GPD):
\begin{equation}
    G_{\xi, \sigma}(y) = 1 - \left( 1 + \frac{\xi y}{\sigma} \right)^{-1/\xi}
\end{equation}
where $\xi$ is the shape parameter and $\sigma$ is the scale parameter. Fitting the upper $2\%$ tail of benign calibration score distributions allows us to solve for operational thresholds corresponding to strict target False Positive Rates (e.g., $\text{FPR} = 0.00\%$).

\section{System Architecture \& Engineering}

Our architecture consists of three modular components:

\subsection{Cross-Packet Stateful Masking (v2)}
To prevent neural memory of transient network identifiers, our pipeline implements stateful cross-packet masking:
\begin{itemize}
    \item Stochastic masking of MAC addresses, IPv4/IPv6 headers, and TCP/UDP port numbers.
    \item Stateful continuation tracking across TCP packet boundaries to preserve TLS record header state.
    \item Masking of QUIC and VPN handshake noise fields.
\end{itemize}

\subsection{MambaBytePatcher Backbone}
The model embeds raw byte tokens into a $d_{\text{model}} = 128$ dimensional space, followed by $N=2$ stacked Mamba-2 blocks with expansion factor $E=2$, head dimension $h_{\text{dim}}=64$, and state dimension $d_{\text{state}}=64$. The final output projects to a 256-way softmax logit layer.

\subsection{Hardware Acceleration & Prefetching Pipeline}
To eliminate I/O bottlenecks during streaming training, our DataLoader incorporates background CPU batch prefetching (\texttt{prefetch\_factor=8}, \texttt{persistent\_workers=True}) alongside PyTorch Distributed Data Parallel (DDP) and native C++/CUDA \texttt{selective\_scan\_cuda} kernels, achieving an accelerated training throughput of $\sim 440\,\text{steps/minute}$ on dual NVIDIA Tesla T4 GPUs.

\begin{itemize}

\subsubsection{Key Findings}
\begin{enumerate}
    \item \textbf{Exponential FPR Decay}: Scaling dataset exposure from $164\,\text{MB}$ to $1.64\,\text{GB}$ drove holdout false positive rates down from $44.06\%$ to \textbf{0.0176\% (only 6 false alarms out of 34,043 windows)}, representing a $2,500\times$ reduction in false alarms and achieving \textbf{99.982\% benign accuracy}.
    \item \textbf{Perfect EVT Operational Bound}: Under EVT thresholding ($10.674\,\text{bits}$), the model achieved \textbf{0.000\% FPR (0 false alarms / 34,043 windows)} while maintaining a \textbf{28.4\% zero-day detection rate}.
    \item \textbf{Peak Attack Separation}: Step 10,000 achieved a peak \textbf{0.9788 Calibration ROC-AUC} and \textbf{94.6\% recall} across 24 attack families.
\end{enumerate}

\section{Limitations \& Operational Scope}
While our framework demonstrates state-of-the-art anomaly separation and zero false-alarm bounds, full operational deployment introduces specific real-world boundaries:
\begin{enumerate}
    \item \textbf{TCP Flow Reassembly}: Evaluated on offline, sequentially captured PCAP streams. Real-world 10\,Gbps line-rate deployment requires a stateful C++/eBPF flow-reassembly engine to handle packet reordering and packet drops across concurrent connections.
    \item \textbf{Payload Encryption (TLS 1.3 \& QUIC)}: For encrypted application payloads, byte-level surprise is concentrated in unencrypted TLS Handshake negotiations (SNI, cipher suites, certificates) and packet length/timing dynamics, unless an inline TLS decryption proxy is deployed.
    \item \textbf{Inference Latency Budget}: Per-window FP16 inference latency ($\approx 1.5\,\text{ms}$) necessitates out-of-band TAP/SPAN port deployment or dedicated SmartNIC/DPU hardware offloading for high-speed perimeter links.
\end{enumerate}

\section{Conclusion}
We have presented the Sovereign Byte Firewall, demonstrating that Mamba-2 Selective State Space Models combined with Extreme Value Theory achieve near-perfect benign accuracy ($99.982\%$) and robust zero-day threat detection without relying on cloud APIs or CVE signatures. Operating with sub-2ms latency, this framework establishes a viable foundation for sovereign, air-gapped cybersecurity perimeters.
